12.ECMAScript 6

2020-01-27 js基础 ECMAScript 6, js基础 0 评论 字数统计: 2.7k(字) 阅读时长: 13(分)

严格模式

开启严格模式，当使用SE6模块化开发时，严格模式不需要写，自动启用

如果不使用var、let、const定义变量和常量就不能直接使用

  • 开启严格模式的方式 “use strict”;

不允许使用with

不能对只读属性赋值

还有哪些只读属性？
  • 冻结的属性也是只读属性
  • 定义后的只读属性
  • 对象中只有get方法，没有set方法，因此，也是只读属性

let && const && 箭头函数

let

定义一个有范围变量。不是window的属性

  • let定义的变量仅限于循环及花括号语句块内

  • let定义的变量仅作用于语句块内

const

常量 不可以修改

  • 不能修改引用地址，但是可以修改对象属性

箭头函数

函数的参数，如果只有一个参数，可以不写括号
函数的参数，如果没有参数或者参数在一个上，就需要加括号

  • 箭头函数中，如果函数内容只有一句话，并且返回该一句话的内容，就可以不用加｛｝，并且不需要写return
  • 如果箭头函数中有多句代码，或者不返回任何内容，就需要加｛｝
  • 箭头函数的用法有哪些
  • 匿名函数中使用函数箭头
  • 箭头函数改变了this的指向

set && Map

数组和对象

数组的优点

1.有顺序
2.紧密性
3.有关联

数组的缺点
1.速度慢，查询，添加，删除
2.有重复
3.判断值时，必须遍历数组

对象的优点
1.有键值对，查询快，插入和删除也很快
2.键唯一
3.多重集合

对象的缺点
1.没有关联
2.按照添加顺序遍历（顺序无法改变）
3.如果需要查询属性时，需要遍历

set

set是一个不能有重复元素的集合，重复添加无效

  • 新建set
  • add(value);
  • delete();
  • has(value);
  • clear();
  • #####键值完全相同
  • for of 遍历
  • 数组去重

WeakSet();

弱引用列表类型

  • 弱引用列表是不可遍历的
  • 当将obj设为null 被维护WeakSet列表中会自动清除它

map 类型

需要有键值对，需要有长度，可以遍历属性列表，可以遍历数值列表，增删盖查速度快
